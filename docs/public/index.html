Compose

Run Helm charts on Docker/Podman without Kubernetes. An open-source tool that lets you take Helm charts and run them directly on Docker or Podman, without needing a Kubernetes cluster. It bridges the gap between Kubernetes-native packaging (Helm) and traditional container runtimes (docker-compose.yaml), giving you the same artifacts and deployment flows across both worlds.

Why choose Compose?

Compose is a lightweight tool that allows you to run Helm charts on Docker or Podman without needing a Kubernetes cluster. Here are some reasons to choose Compose:

cloud_off

No Kubernetes Required

Deploy your applications on Docker or Podman without setting up a Kubernetes cluster. Perfect for edge devices, RHEL environments, and resource-constrained systems where Kubernetes isn’t available.
